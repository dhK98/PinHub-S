# PinHub-S (Server)

An application that creates notes based on a map

## Characteristics

- MSA Programming for scalability
- test-driven development
- for Handling High-Capacity Traffic

## Environment

- Language: TypeScript
- Server Engine: Node.js(v20.11.1)
- DataBase: Postgres

## branch process

- feature-{feature name}
  ->feature
  -> develop
  -> main
  -> release
